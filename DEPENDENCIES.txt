INSTALL ALL PACKAGES
─────────────────────
sudo pacman -S i3-gaps polybar alacritty rofi feh thunar \
  mpd mpc ncmpcpp picom papirus-icon-theme \
  ttf-jetbrains-mono-nerd noto-fonts \
  pulseaudio pulseaudio-alsa pavucontrol \
  network-manager-applet scrot lxappearance

From AUR:
yay -S starship arc-gtk-theme

─────────────────────
PACKAGE PURPOSE
─────────────────────
i3-gaps           Window manager
polybar           Status bar
alacritty         Terminal
rofi              App launcher
feh               Wallpaper setter
thunar            File manager
mpd               Music daemon
mpc               Music controller
ncmpcpp           Music player TUI
picom             Compositor
papirus-icon-theme  Icons
ttf-jetbrains-mono-nerd  Font
noto-fonts        Symbols font
pulseaudio        Audio
pavucontrol       Volume GUI
network-manager-applet  Network
scrot             Screenshots
starship          Shell prompt
arc-gtk-theme     GTK theme
lxappearance      Theme switcher

─────────────────────
INSTALLATION STEPS
─────────────────────
1. Install packages above
2. Copy config/* to ~/.config/
3. Copy starship.toml to ~/.config/
4. Copy .bashrc to ~/
5. Make scripts executable:
   chmod +x ~/.config/polybar/launch.sh
   chmod +x ~/.config/polybar/playpause.sh
   chmod +x ~/.config/rofi/powermenu.sh
   chmod +x ~/.config/i3/init-workspaces.sh
6. Add wallpaper to ~/wallpapers/wall.jpg
7. Add music to ~/Music/
8. Log into i3 session
9. Run: mpc update && mpc add / && mpc play
